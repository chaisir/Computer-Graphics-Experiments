# Computer Graphics Experiments
## 重庆大学计算机图形学实验

### 画直线——line.ipynb

* 采用Jupyter notebook，使用了matplotlib库，在坐标系中打点，但打出的点并不是单个像素点，而是宏观意义上上的一个点。

* 三种方法：
 * 数值微分法
 * 中点画线法
 * Bresenham算法
    
### 画圆——circle.ipynb

* 采用Jupyter notebook，使用了matplotlib库，在坐标系中打点，但打出的点并不是单个像素点，而是宏观意义上上的一个点。

*  圆：
 * 中点画圆
 * Bresenham算法画圆

* 椭圆：
 * 中点法画椭圆
